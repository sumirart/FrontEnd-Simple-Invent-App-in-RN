# FrontEnd-Simple-Invent-App-in-RN
Simple React Native invent app that can communicate with REST backend

Backend: https://github.com/sumirart/Backend-Simple-Invent-App-in-Express

## Installation
1. Clone project
```
git clone https://github.com/sumirart/FrontEnd-Simple-Invent-App-in-RN.git
```

2. cd into folder
```
cd frontend-simple-invent-app-in-rn
```

3. Download dependencies
```
npm install
```

4. Find IP and change it to your backend IP
```
const IP = "http://your-backend-IP:port/user/";
```

5. Run
```
react-native run-android
```
